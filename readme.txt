== Description ==

Integriere hochwertige **Vergleiche oder Widgets für Mobilfunk- und DSL-Tarife in Deutschland** mit dem TARIFFUXX-Plugin:
- Einfache Integration via Shortcode
- Individualisierbar (Einstellungen und Farben)
- Mehrwert für deine Besucher
- Verdiene Provisionen für Tarifvermittlungen

== Installation ==

1. In deinem WordPress klicke auf **Plugins** >>> **Installieren**.
2. Suche nach **TARIFFUXX**.
3. Klicke auf den Button **Jetzt installieren**.
4. Gehe zum neuen Menü **TARIFFUXX Vergleiche & Widgets**.
5. Klicke auf **Neu anlegen** um deine erste Vergleichstabelle oder dein erstes Widget zu erstellen.

*English:*

1. *Visit **Plugins** >>> **Add new** within your WordPress installation.*
2. *Search for **TARIFFUXX**.*
3. *Activate **TARIFFUXX** by installing from your **Plugins** menu.*
4. *Visit new **TARIFFUXX Vergleiche & Widgets** menu.*
5. *Click **Neue anlegen** to add a new comparison table or widget.*
